## 🔄 更新方式

### 自动更新（推荐）
1. 在系统工作台点击"有新版本！"按钮
2. 查看更新详情后点击"立即更新"
3. 系统将自动下载并安装更新

### 手动更新
```bash
# 方式1：使用更新脚本
./update.bat

# 方式2：Git手动更新
git fetch origin
git pull origin main
cd ruoyi-ui
npm install
npm run build:prod
```
